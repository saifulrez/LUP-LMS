Attribute The starting tag of an element can have special snippets of Information we can attach, called attribute Attributes have the key=”value” syntax:
< p class=”a-class”>A paragraph of text< /p> You can also use single quotes, but using double quotes in HTML Is a nice convention. We can have many of them: < p class=”a-class” id=”an-id”>A paragraph of text< /p> And some attributes are Boolean, meaning you only need the key: < script defer src=”file.js”>< /script> The class and id attributes are two of the most common you will Find used.
They have a special meaning, and they are useful both in CSS and Javascript. The difference between the two is that an id is unique in the Context of a web page; it cannot be duplicated.
